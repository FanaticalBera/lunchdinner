## 🔭 다음 스텝 — 뭘 하면 좋을까?

### 🥇 **Priority 1: 퀄리티 마무리 (지금 당장 건드릴 것들)**

#### 1-A. 🎯 빠른 추천 알고리즘 고도화
지금 `pickQuickRecommendation`은 **태그 매칭 수 기준으로 동점 풀에서 랜덤 픽**하는 구조야.  
근데 "오늘 기분은 든든하고 매운 거"를 골랐을 때 더 **정밀하게** 골라주면 좋겠지?

**할 수 있는 것들:**
- **태그 우선순위 가중치**: 유저가 더 중요하게 생각하는 태그 순서 반영
- **최근 추천 히스토리 제외**: 같은 메뉴 연속 추천 방지 (세션 내)
- **"이건 싫어" 제외 기능**: 결과 화면에서 "이거 말고" 버튼 추가

#### 1-B. 🔤 태그 선택 UX 개선 (`QuickTagScreen`)
지금은 태그를 **다 동등하게** 나열하고 있는데, 태그 수가 이미 많음.

**할 수 있는 것들:**
- 태그 카테고리 그룹핑 (식감 / 나라별 / 분위기)
- 태그 선택 순서 표시 (1순위, 2순위 느낌으로)
- `0개 선택 → 전체에서 랜덤` 안내 문구 추가

#### 1-C. 🎬 화면 전환 애니메이션 통일
스텝 이동 시 **슬라이드 방향**이 일관되지 않은 부분이 있어.  
`AppShell`에서 `AnimatePresence` + `key={guardedStep}` 조합으로 통일 가능.
